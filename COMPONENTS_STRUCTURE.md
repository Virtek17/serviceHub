# Структура компонентов

## Созданные компоненты

### Common (Общие)

- `Logo.jsx` - Логотип приложения
- `PageHeader.jsx` - Универсальный хедер с навигацией, логотипом и информацией о пользователе
- `EmptyState.jsx` - Компонент для отображения пустых состояний

### Dashboard (Дашборд мастера)

- `DashboardSidebar.jsx` - Боковое меню дашборда
- `OverviewTab.jsx` - Вкладка обзора со статистикой
- `ServicesTab.jsx` - Вкладка управления услугами
- `StatCard.jsx` - Карточка статистики
- `BookingCard.jsx` - Карточка записи клиента
- `ServiceCard.jsx` - Карточка услуги
- `TimeSlotCard.jsx` - Карточка временного слота

### Filters (Фильтры)

- `FilterSidebar.jsx` - Боковая панель с фильтрами
- `SearchInput.jsx` - Поле поиска
- `SelectFilter.jsx` - Выпадающий список фильтра
- `TagFilter.jsx` - Фильтр по тегам

### Profile (Профиль мастера)

- `ProviderInfo.jsx` - Информация о мастере
- `ServicesList.jsx` - Список услуг мастера
- `ServiceItem.jsx` - Элемент услуги
- `TimeSlotsList.jsx` - Список доступных слотов

### Modals (Модальные окна)

- `BookingModal.jsx` - Модальное окно бронирования

## Преимущества новой структуры

1. **Переиспользование** - компоненты можно использовать в разных местах
2. **Читаемость** - страницы стали в 3-4 раза короче
3. **Тестируемость** - каждый компонент можно тестировать отдельно
4. **Поддержка** - легче найти и исправить баги
5. **Масштабируемость** - проще добавлять новые функции
